## Responsive Container

The `Container` responsively sets a `max-width` for each configured breakpoint equal to `calc(breakpoint - gutter)`.

<Canvas withToolbar>
  <GridConfigProvider value={gridConfig}>
    <Container>
      <Box>Container</Box>
    </Container>
  </GridConfigProvider>
</Canvas>

## Fluid Container

When used with the `fluid` prop, `max-width` is always `100%`.

<Canvas withToolbar>
  <GridConfigProvider value={gridConfig}>
    <Container fluid>
      <Box>Fluid Container</Box>
    </Container>
  </GridConfigProvider>
</Canvas>
